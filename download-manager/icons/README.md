# 图标文件说明

请在此文件夹中放置以下尺寸的图标文件：

- `icon16.png` - 16x16 像素
- `icon32.png` - 32x32 像素  
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 图标要求

- 格式：PNG（支持透明背景）
- 建议使用下载相关的图标设计
- 可以使用在线图标生成工具或设计软件创建

## 临时解决方案

如果暂时没有图标文件，可以：
1. 从网上下载免费的下载图标
2. 使用在线图标生成工具（如 favicon.io）
3. 或者暂时使用纯色占位图片

插件在没有图标的情况下也能正常工作，只是浏览器工具栏上会显示默认图标。
