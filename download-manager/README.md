# 下载管理器 Chrome 插件

一个模拟 Edge 浏览器下载管理器的 Chrome 插件，可以显示下载历史记录和文件删除状态。

## 功能特性

- 📥 显示所有下载历史记录
- 🗑️ 检测文件是否已删除
- 🔍 搜索下载记录
- 📂 快速打开下载文件夹
- 🎨 macOS 风格的玻璃质感界面
- 📱 响应式设计

## 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `download-manager` 文件夹
5. 插件安装完成后，点击浏览器工具栏上的插件图标即可使用

详细安装说明请查看 [INSTALL.md](./INSTALL.md)

## 使用说明

1. **查看下载记录**：点击浏览器工具栏上的插件图标，即可查看所有下载记录
2. **搜索文件**：点击搜索图标，输入文件名进行搜索
3. **打开文件**：点击文件项或"打开文件"链接即可打开文件（仅限未删除的文件）
4. **打开下载文件夹**：点击文件夹图标打开默认下载文件夹

## 技术实现

- 使用 Chrome Downloads API 获取下载历史
- 通过 `downloads.exists` 属性检测文件是否存在
- 使用 Chrome Downloads API 打开文件
- macOS 风格的玻璃质感 UI（glassmorphism）

## 注意事项

- 需要 Chrome Downloads 权限才能访问下载记录
- 文件删除检测基于 Chrome 的 `exists` 属性，可能不完全准确
- 建议定期清理已删除的文件记录
